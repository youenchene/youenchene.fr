Avec Laurent Deséchalliers nous organisons, une première terrasse vendredi 1 août sur rouen à partir de 18h30.

A l'image des Entreprenautes Associées de ce soir (Soirée OFF), il s'agit de se rencontrer autour de sujets tels que les Technologies de l'Information, le web l'entreprenariat, etc ...

Pour connaître le lieu, il suffit de suivre le twitter LesTerrassesIT, Laurent doit vous indiquera le bar en question (il doit donner les résultat de son "repérage" de ce week end.

Pour indiquer votre venu, vous pouvez répondre au Twitter (@LesTerrassesIT), laisser un message sur ce blog, sur le groupe facebook ou tout simplement venir "à la fraîche".

A vendredi pour voir si le monde IT rouennais peut frémir dans le début de chaleur du mois d'Août.

EDIT: Pour le lieu, cela sera L'échiquier (Bar à coté de l'espace du palais / la FNAC).
