Vous êtes une boutique en ligne, vous assurez la partie livraison et surtout vous vendez des produits qui viennent directement de chez les producteurs.

Nous vous proposons d'afficher ces produits sur Fermiers d'à côté!


Pour cela, vous avez 3 possibilités :
  • vous vous inscrivez en tant que producteur revendeur et vous ajoutez vos produits manuellement via l'interface de fermiers d'à côté,
  • vous n'avez pas la main sur la partie informatique de votre boutique nous vous proposerons une interface qui vous permettra d'afficher vos produits avec lien pointant vers la description du produit dans votre boutique,
  • vous avez la main sur la partie informatique de votre boutique nous vous proposons de synchroniser une partie de votre base de produits, ces produits seront affichés avec lien pointant vers la description du produit dans votre boutique.
Pour toutes informations complémentaires, contactez moi : youen@fermiersdacote.fr.
